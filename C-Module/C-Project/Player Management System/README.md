# Cricket Player Management System (C)

This is a console-based Cricket Player Management System written in C.

## Features
- Add new players using dynamic memory allocation
- Remove player by jersey number
- Search player details
- Update player records
- Sort players by runs and wickets
- Display top 3 players
- Menu-driven program

## Concepts Used
- Structures
- Pointers
- malloc() and realloc()
- Searching and sorting
